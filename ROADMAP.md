# 🗺 RAGForge – Kubernetes-Native Development Roadmap

This roadmap evolves from basic Hybrid RAG to full AI Infrastructure system running on Kubernetes.

---

# 🟢 Stage 1 – Core Hybrid RAG (Minikube)

Dataset:
Kubernetes Basics Printable Version (~24 pages)

Source:
https://kubernetes.io/docs/tutorials/kubernetes-basics/_print/

Goals:
- Extract PDF text
- Clean & normalize
- Apply chunking
- Generate embeddings
- Deploy Vector DB Pod
- Deploy RAGForge API Pod
- Grounded responses with citation

Deliverable:
Fully functional Hybrid RAG running inside Minikube.
